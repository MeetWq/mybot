### 简介

基于 [NoneBot](https://github.com/nonebot/nonebot2) 的QQ机器人

实现了一些乱七八糟的功能

本项目是自用bot项目，并不是开箱即用的，但可以提供某些插件功能上的思路，有问题也可以提issue。
